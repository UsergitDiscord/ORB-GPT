## 🛠️ Tech Stack

- **Frontend:** React + TypeScript + Vite
- **UI:** Tailwind CSS + shadcn/ui components  
- **AI:** Google Gemini API (text + image generation)
- **PWA:** Service Worker + Web App Manifest
- **Hosting:** GitHub Pages (free!)
